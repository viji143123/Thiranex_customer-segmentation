Customer Segmentation Project

Objective

The objective of this project is to segment customers based on their purchasing behavior using K-Means Clustering.

Tools Used

- Python
- Pandas
- Matplotlib
- Scikit-Learn
- VS Code

Dataset

Mall Customers Dataset

Features

- Customer ID
- Gender
- Age
- Annual Income (k$)
- Spending Score (1-100)

Methodology

1. Imported the dataset using Pandas.
2. Selected Annual Income and Spending Score as features.
3. Applied K-Means Clustering algorithm.
4. Segmented customers into 5 groups.
5. Visualized the customer segments using a scatter plot.

Output

A scatter plot was generated showing different customer segments based on Annual Income and Spending Score.

Result

Customers were successfully segmented into different groups. This helps businesses identify customer behavior patterns and improve marketing strategies.

Conclusion

Customer segmentation using K-Means Clustering is an effective method for analyzing customer purchasing behavior and identifying target customer groups.
